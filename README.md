## 🚀 快速開始

### 前置需求

- Claude AI 訪問權限
- VS Code 或其他支援 Claude Code 的編輯器
- 基本的 AI 提示詞使用知識

### Claude Code 使用者

1. **複製提示詞**
   ```bash
   # 選擇你喜歡的提示詞
   cat prompts/traditional_chinese/tsundere_maid_coder.md
   ```

2. **應用到 Claude.md**
   ```bash
   # 複製到你的 Claude 設定檔
   cp prompts/traditional_chinese/tsundere_maid_coder.md ~/.claude/CLAUDE.md
   ```

3. **重啟 Claude Code 享受新體驗！**
